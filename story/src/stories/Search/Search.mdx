# Search

The Search component provides a text input with search functionality, supporting both left-to-right (LTR) and right-to-left (RTL) layouts. It includes a search icon and an optional clear button when the input contains text.

## Features

- **Multiple States**: idle, hover, active, focused, and disabled
- **RTL Support**: Full right-to-left layout support for international use
- **Clear Functionality**: Automatic clear button when input has content
- **Keyboard Accessible**: Full keyboard navigation and focus management
- **Material-UI Icons**: Uses MUI Search and Close icons
- **Focus Ring**: Visual focus indicator for accessibility
