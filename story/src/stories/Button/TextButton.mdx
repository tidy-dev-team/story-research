# Text Button Component

The Text Button component is a minimal UI element for triggering secondary actions or navigation without drawing too much attention away from the main content of the interface. It's presented as a simple text link with blue color.

## Features

- Clean, text-only design with a blue color.
- Support for leading and trailing icons.
- RTL (right-to-left) text direction support.
- Accessible focus states.
- Disabled state.
- Click event handling.

## Import

```tsx
import { TextButton } from './path/to/TextButton';
import LanguageIcon from '@mui/icons-material/Language'; // Example Icon
```

## Basic Usage

```tsx
// Basic text button
<TextButton label="Learn more" />

// With click handler
<TextButton 
  label="Visit FAQ" 
  onClick={() => console.log('Navigate to FAQ')}
/>
```

## With Icons

Text buttons can include leading and/or trailing icons.

```tsx
// With leading icon
<TextButton 
  label="Visit Website" 
  leadingIcon={<LanguageIcon sx={{ fontSize: "14px" }} />}
/>

// With trailing icon
<TextButton 
  label="Next Page" 
  trailingIcon={<ArrowForwardIcon sx={{ fontSize: "14px" }} />}
/>

// With both icons
<TextButton 
  label="Open Audio" 
  leadingIcon={<HeadphonesIcon sx={{ fontSize: "14px" }} />}
  trailingIcon={<ArrowForwardIcon sx={{ fontSize: "14px" }} />}
/>
```

## States

### Disabled

When disabled, the text button is non-interactive and its appearance changes to indicate this state.

```tsx
<TextButton label="Disabled Action" disabled />
```

### Focus

The text button shows a focus ring when it receives keyboard focus or when the `focused` prop is true.

```tsx
<TextButton label="Focused Action" focused />
```

## RTL Support

For right-to-left languages, set the `rtl` prop to `true`. Icons will also be flipped accordingly if applicable.

```tsx
<TextButton 
  label="פעולה" // Example RTL text
  rtl={true} 
  leadingIcon={<LanguageIcon sx={{ fontSize: "14px" }} />}
/>
```

## Event Handling

The `onClick` prop can be used to define the action performed when the button is clicked.

```tsx
<TextButton 
  label="Submit Form" 
  onClick={(event) => {
    console.log('Submit button clicked', event);
    // Perform submission logic here
  }}
/>
```
